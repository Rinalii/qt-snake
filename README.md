# Qt Snake
## Проект для портфолио | C++17 | QT

# Классическая игра «Змейка», реализованная на Qt и C++ с использованием архитектурного паттерна Model‑View‑Controller (MVC). Игровой процесс сопровождается плавной анимацией, счётом, паузой и меню перезапуска.

## ⚙️ Сборка и запуск
### Требования

- C++17
- QT 6.8

## 🧱 Возможности

+ Управление змейкой – клавиши-стрелки (↑ ↓ ← →).
+ Сбор еды с увеличением длины змейки.
+ Пауза и продолжение – по нажатию на пробел игра ставится на паузу, появляется меню с кнопками "продолжить" и "сброс".
+ Окончание игры – при столкновении с собой отображается сообщение о поражении, доступен рестарт.
+ Визуальные детали – голова змейки имеет «глаза», направленные в сторону движения; сегменты тела уменьшаются ближе к хвосту.
